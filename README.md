# React Structurizer MVP

App.jsx 코드를 붙여넣으면 Vite + React 프로젝트 ZIP을 생성합니다.

## 실행 방법 (필수)

**파일을 더블클릭해서 열면 CDN 스크립트가 로드되지 않아 동작하지 않습니다.**  
반드시 로컬 서버로 실행한 뒤 브라우저에서 접속하세요.

```bash
npx serve . -p 3000
```

이후 브라우저에서 **http://localhost:3000** 으로 접속합니다.

또는:

```bash
npm start
```

## 사용법

1. Project Name 입력
2. App.jsx 코드를 텍스트 영역에 붙여넣기
3. "Generate ZIP" 클릭 → ZIP 파일 다운로드
