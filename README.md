# React Structurizer MVP

App.jsx 코드를 붙여넣으면 Vite + React 프로젝트 ZIP을 생성합니다.

## 실행 방법 (필수)

**파일을 더블클릭해서 열면 CDN 스크립트가 로드되지 않아 동작하지 않습니다.**  
반드시 로컬 서버로 실행한 뒤 브라우저에서 접속하세요.

```bash
npx serve . -p 3000
```

이후 브라우저에서 **http://localhost:3000** 으로 접속합니다.

또는:

```bash
npm start
```

## 사용법

1. Project Name 입력
2. App.jsx 코드를 텍스트 영역에 붙여넣기
3. "Generate ZIP" 클릭 → ZIP 파일 다운로드

Zip 파일을 압축해제 한후에 실행.bat는 마우스 오른클릭로 맨아래 차단해제를해야 합니다. 
"실행 파일(실행.bat) 사용 시 필수" 섹션 추가
실행 파일을 만들었을 때(압축 해제 후 처음 실행할 때)에는 bat 파일을 반드시 차단 해제해야 한다고 안내
실행.bat에서 마우스 오른쪽 클릭 → 속성 → 일반 탭 맨 아래 "차단 해제" 체크 → 확인 → 그 다음 실행.bat 더블클릭
"파일을 복사할 수 없음" / Windows 보안 경고 섹션
위와 같이 실행.bat 우클릭 → 속성 → 차단 해제 후 다시 실행하라고 안내
